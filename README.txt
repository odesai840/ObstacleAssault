# ObstacleAssault
Assets and C++ code from ObstacleAssault Unreal Engine project

ObstacleAssault is a project created using Unreal Engine 5. This project contains
multiple components coded in C++. Movement values for these components need to be
set using Unreal Engine on startup. Gameplay consists of a short obstacle course
with various obstacles that move up and down, move side to side, and rotate.

Controls: WASD for movement
